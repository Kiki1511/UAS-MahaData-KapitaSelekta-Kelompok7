## Cara Menjalankan WebGIS
1. Buka repository GitHub ini dan unduh (Download ZIP) atau lakukan `git clone`.
2. Buka folder `webgis` menggunakan Visual Studio Code.
3. Jalankan file `index.html` menggunakan ekstensi **Live Server** (atau akses melalui GitHub Pages jika sudah tersedia).
