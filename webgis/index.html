Kelompok 7
Google Earth Engine

Platform geospasial berbasis cloud dari Google untuk analisis data citra satelit berskala besar secara efisien.

Sentinel-2

Satelit observasi bumi dari ESA yang menyediakan citra multispektral resolusi tinggi untuk pemantauan tutupan lahan.

NDVI

Normalized Difference Vegetation Index — indeks untuk mengukur tingkat kehijauan dan kesehatan vegetasi dari citra satelit.

Random Forest

Algoritma machine learning ensemble yang menggunakan banyak decision tree untuk klasifikasi tutupan lahan dengan akurasi tinggi.

Change Detection

Teknik analisis perubahan tutupan lahan antar dua periode waktu untuk mengidentifikasi area gain dan loss vegetasi.

Leaflet & WebGIS

Library JavaScript open-source untuk membuat peta interaktif yang digunakan sebagai platform visualisasi hasil analisis geospasial.
